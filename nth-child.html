HTML > :nth-child()

Псевдокласс :nth-child() используется для выбора необходимых элементов из списка, вот пример его использования: 

.example tr:nth-child(2n+1) {
      background:  lightgrey;
}
в указанном примере выбирается каждый 2-ой элемент tr , начиная с самого первого его элемента и изменяет фоновый цвет этих элементов на светло-серый цвет.

Приницип работы :nth-child() следующий.
В скобках указывается параметр (an+b) - где
a - является элемент с которого мы начинаем;
n - является счётчиком и начинает свой счёт с 0 до бесконечности;
b - является число которое мы прибавляем для перехода к следующему элементу;

Итого, параметр (2n+1) означает следующее:

a n b Расчёт
2 0 1 (2 * 0) + 1 = 1
2 1 1 (2 * 1) + 1 = 3
2 2 1 (2 * 2) + 1 = 5

стоит обратить внимание на работу с отрицательным счётчиком n. Разберём пример (-n+3)

a -n b Расчёт
- 0 3 0 + 3 = 3
- -1 3 -1 + 3 = 2
- -2 3 -2 + 3 = 1
